# Kongoni Bridge
A bridge-building game made for [Ludum Dare 36][1]

![a truss spanning a chasm](assets/screenshot.png)

## How to Build
Compiling from source requires cmake, qt5 and a compiler that supports C++14.

    cmake -H. -Bbuild && cmake --build build
